еҸ‘иЎЁдәҺ 2017-10-23 | еҲҶзұ»дәҺ CS | |   |   483 words

еҶҷеңЁеүҚйқў

еңЁйғЁзҪІзҪ‘з«ҷеҗҺйҒҮеҲ°зҡ„дёҖдёӘеӨ§еқ‘пјҢй•ҝиҝҷж ·пјҡ

зңӢеҲ°иҝҷдёҖиЎҢзҶҹжӮүзҡ„weвҖҷre sorry, but something went wrongпјҢй»ҳй»ҳи·ҹиҝҷдҪҚиҖҒжңӢеҸӢжү“дёӘжӢӣе‘јгҖҗжҚӮи„ёгҖ‘гҖӮ

GoogleеҗҺпјҢеҸ‘зҺ°е…¶д»–з”ЁpassengerжқҘйғЁзҪІRailsзҡ„пјҢд№ҹйҒҮеҲ°иҝҮиҝҷз§ҚйЎөйқўж— жі•жҳҫзӨәзҡ„й—®йўҳпјҢдҪҶжҳҜжІЎзңӢеҲ°жңүдәәжҸҗдҫӣи§ЈеҶіж–№жі•жҲ–иҖ…жҖқи·Ҝзҡ„пјҢеҶіе®ҡи®°еҪ•дёӢжҲ‘зҡ„ж•ҙдёӘи§ЈеҶіжӯҘйӘӨпјҢз»ҷеҗҢж ·е…Ҙеқ‘зҡ„дјҷдјҙд»¬дҪңдёӘеҸӮиҖғгҖӮ

жӯЈж–Ү

GoogleпјҢзңӢдәҶstack overflowдёҠзҡ„дёҖдәӣзӣёе…іеҶ…е®№еҗҺпјҢзҢңжөӢжҳҜpassengerзҡ„й—®йўҳпјҢдәҺжҳҜз»Ҳз«Ҝиҫ“е…Ҙsudo passenger-status, еҸ‘зҺ°passengerж №жң¬е°ұжІЎжңүејҖиҝҷдёӘRailsзҡ„иҝӣзЁӢгҖӮ

жІЎејҖйӮЈе°ұжүӢеҠЁжқҘејҖпјҢеңЁappзҡ„currentзӣ®еҪ•дёӢпјҢиҝҗиЎҢbundle exec passenger start, жҠҘй”ҷпјҡ

жҖҺд№ҲеҠһпјҹжҢүз…§жҢҮзӨәеҺ»tmpдёӢжүҫж–Үд»¶пјҢзңӢerror logгҖӮ

д»Ҙrootиә«д»Ҫзҷ»еҪ•пјҢеҺ»tmpзӣ®еҪ•дёӢпјҢжҹҘзңӢеҮәй”ҷзҡ„log,иҝҷйҮҢжҳҜpassenger-error-pxhJuZ.htmlгҖӮ

жү“ејҖжҳҫзӨәеҰӮдёӢпјҡ

OKпјҢйӮЈе°ұеҠЁеҠЁlisten иҝҷдёӘgem, google еҗҺпјҢеҸӮиҖғstackoverflowдёҠзҡ„и§Јзӯ”LoadError: Could not load the вҖҳlistenвҖҷ gem (Rails 5)пјҢ жҠҠlistenд»Һdevelopmentдёӯз§»йҷӨпјҢж”ҫеңЁдәҶglobalдёӯ

жң¬ең°иҝҗиЎҢbundleпјҢgit commit е’Ң git push еҗҺпјҢиҝӣиЎҢйғЁзҪІ cap production deploy, еҲ·ж–°йЎөйқўпјҢеҸ‘зҺ°иҝҳжҳҜиҖҒж ·еӯҗгҖӮжҖҺд№ҲеӣһдәӢпјҹзӘҒз„¶жғіеҲ°stack overflowдёҠжҸҗеҲ°еҸҜд»Ҙзҷ»еҪ•rails consoleзңӢзңӢпјҢеҘҪпјҢиҜ•иҜ•гҖӮ

з”Ёdeployиә«д»Ҫзҷ»еҪ•иҝңз«ҜеҗҺпјҢжү§иЎҢпјҡ

жһңз„¶пјҢжҠҘй”ҷпјҢжҳҫзӨәзҪ‘з«ҷжңүbugпјҢз«Ӣй©¬еӣһеҺ»дҝ®bugвҖҰвҖҰ

еңЁжң¬ең°дҝ®еӨҚеҘҪпјҢgit commit е’Ң git push еҗҺпјҢиҝӣиЎҢйғЁзҪІ cap production deploy, еҲ·ж–°йЎөйқўпјҢ OKпјҒпјҒ

The End

иҝҷж®өзҲ¬еқ‘зҡ„ж—ҘеӯҗпјҢйҒҮеҲ°дәҶдёҚе°‘GoogleдәҶд№ҹдёҚдёҖе®ҡиғҪи§ЈеҶізҡ„еқ‘пјҢз”ҡиҮідјҡи®©дҪ еңЁжҮөйҖјзҡ„жғ…еҶөдёӢпјҢиў«еҲ«дәәзҡ„и§Јзӯ”еёҰеҫ—и¶ҠжқҘи¶ҠиҝңгҖӮеҸӘжғіиҜҙи§ЈеҶій—®йўҳзҡ„жҖқи·Ҝзңҹзҡ„еӨӘйҮҚиҰҒдәҶпјҢеҗҢж—¶еӨҡзӮ№иҖҗеҝғпјҢзӣёдҝЎеқ‘еқ‘зҡҶжңүи§ЈгҖӮ

Be more patient.
