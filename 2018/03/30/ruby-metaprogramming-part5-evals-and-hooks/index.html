иҜ»гҖҠRubyе…ғзј–зЁӢгҖӢд№Ӣevalе’Ңhooks

еҸ‘иЎЁдәҺ 2018-03-30 | еҲҶзұ»дәҺ Reading |   |   1,467 words

гҖҠRubyе…ғзј–зЁӢгҖӢз¬¬6з« зј–еҶҷд»Јз Ғзҡ„д»Јз Ғзҡ„еӯҰд№ з¬”и®°пјҢдё»иҰҒеҶ…е®№жҳҜevalе’Ңй’©еӯҗж–№жі•гҖӮ

д»ҺйңҖжұӮиҜҙиө·

жң¬з« ејҖе§Ӣз”ЁдёҖдёӘbossжҸҗеҮәзҡ„й—®йўҳдҪңдёәеј•еӯҗпјҢжҲ‘и§үеҫ—иҝҷдёӘдҫӢеӯҗеҫҲеҘҪгҖӮз®Җиҝ°дёӢй—®йўҳпјҡ

еҲӣе»әдёҖдёӘеҗҚдёәattr_checkedзҡ„зұ»е®ҸпјҢиҝҷдёӘзұ»е®ҸиҰҒж»Ўи¶ідёӨдёӘжқЎд»¶пјҡ

  • жҺҘеҸ—еұһжҖ§еҗҚе’Ңд»Јз Ғеқ—пјҢд»Јз Ғеқ—з”ЁдәҺйӘҢиҜҒеұһжҖ§еҖјзҡ„жңүж•ҲжҖ§
  • еҸӘжңүеҪ“зұ»еҢ…еҗ«жҹҗдёӘжЁЎеқ—пјҢжҜ”еҰӮcheckedAttributesж—¶пјҢжүҚеҸҜд»ҘдҪҝз”Ёattr_checked

и§ЈеҶіжҖқи·ҜеҸҜд»ҘиҪ¬еҢ–жҲҗпјҡ

  • еҰӮдҪ•е®ҡд№үдёҖдёӘзұ»е®Ҹж–№жі•пјҹгҖҗе…Ҳз”Ёevalе®ҡд№үдёҖдёӘеҶ…ж ёж–№жі•пјҢз„¶еҗҺеҶҚж”№йҖ жҲҗзұ»е®ҸгҖ‘
  • еҰӮдҪ•з»ҷдёҖдёӘзұ»е®Ҹж–№жі•ж·»еҠ д»Јз Ғеқ—пјҹгҖҗmethodж·»еҠ &blockеҸӮж•°еҚіеҸҜгҖ‘
  • е®ҡд№үжЁЎеқ—checkedAttributesпјҢйҖҡиҝҮй’©еӯҗж–№жі•дёәжҢҮе®ҡзҡ„зұ»ж·»еҠ attr_checked

и®ӨиҜҶдёӢeval

е®ҡд№үиҮӘе·ұзҡ„зұ»е®Ҹж–№жі•еүҚпјҢе…ҲжқҘи®ӨиҜҶдёӢKernel#evalгҖӮ

evalжҳҜеҶ…ж ёж–№жі•пјҢеҸӮж•°жҳҜдёҖж®өrubyд»Јз Ғж–Үжң¬гҖӮзӣёжҜ”instance_evalпјҢ class_evalпјҢе®ғеҸӘиғҪжү§иЎҢд»Јз Ғеӯ—з¬ҰдёІгҖӮйӮЈinstance_evalпјҢ class_evalжҳҜеҗҰиғҪжү§иЎҢд»Јз Ғеӯ—з¬ҰдёІпјҹеҸҜд»ҘгҖӮ

зңӢдёӘдҫӢеӯҗпјҡ

my_array = [1,2,3]
my_array.instance_eval "self.reduce(&:+)" #= > 6
eval("my_array.inject {|sum, x| sum + x}") #= > 6

иҝҷзұ»д»Јз Ғеӯ—з¬ҰдёІеҸҜд»ҘжҗәеёҰдёҖдёӘbindingеҜ№иұЎпјҢз„¶еҗҺеңЁиҜҘеҜ№иұЎзҡ„дҪңз”Ёеҹҹдёӯжү§иЎҢд»Јз ҒгҖӮ

  • evalдёҺbinding

    BindingжҳҜдёҖдёӘз”ЁеҜ№иұЎиЎЁзӨәзҡ„дҪңз”ЁеҹҹпјҢеҸҜд»ҘйҖҡиҝҮKernel#binding жқҘеҲӣе»әгҖӮruby иҝҳжҸҗдҫӣдәҶTOPLEVEL_BINDINGзҡ„йў„е®ҡд№үеёёйҮҸпјҢиЎЁзӨәйЎ¶зә§дҪңз”Ёеҹҹзҡ„BindingеҜ№иұЎгҖӮ

    зңӢд»Јз ҒзҗҶи§ЈдёӢпјҡ

    class MyClass
  def my_method
    @x = 1
    binding
  end
end
obj = MyClass.new.my_method
eval("@x",obj) #= > 1
eval("self", TOPLEVEL_BINDING) #= > main

  • evalзҡ„йә»зғҰ

    жңҖеӨ§зҡ„й—®йўҳжҳҜе®үе…ЁжҖ§пјҢеӣ дёәжү§иЎҢзҡ„жҳҜдёҖж®өеӯ—з¬ҰдёІпјҢrubyжҳҜдёҚдјҡеҜ№еӯ—з¬ҰдёІиҝӣиЎҢиҜӯжі•жЈҖжҹҘзҡ„пјҢе®№жҳ“еј•еҸ‘д»Јз ҒжіЁе…Ҙж”»еҮ»гҖӮ

    зңӢдёӘдҫӢеӯҗпјҡ

    def explore_array(method)
  code = "['a','b','c'].#{method}"
  eval code
end

explore_array("include?('a')") #= > true
explore_array("size") #= > 3
explore_array("object_id; Dir.glob('*')") #= > еҲ—еҮәдәҶиҜҘж–Үд»¶жүҖеңЁзӣ®еҪ•дёӢзҡ„жүҖжңүж–Үд»¶

    иҝҷйҮҢпјҢиҝҗиЎҢзұ»дјјexplore_array("object_id; Dir.glob('*')")иҝҷж ·зҡ„жҒ¶ж„Ҹд»Јз ҒеҸҜиғҪдјҡеёҰжқҘж„ҸжғідёҚеҲ°зҡ„еҗҺжһңгҖӮжүҖд»ҘпјҢиғҪз”Ёд»Јз Ғеқ—е°ұз”Ёд»Јз Ғеқ—гҖӮ

    еҰӮдҪ•йҳІжӯўд»Јз ҒжіЁе…Ҙж”»еҮ»пјҹ

    • и§ЈжһҗжүҖжңүзҡ„д»Јз Ғеӯ—з¬ҰдёІпјҢдёҚзҺ°е®һ

    • ж”№з”ЁеҠЁжҖҒж–№жі•е’ҢеҠЁжҖҒжҙҫеҸ‘жқҘжӣҝжҚўeval

      жҜ”еҰӮдёҠйқўзҡ„дҫӢеӯҗпјҢexplore_arrayеҸҜд»Ҙиҝҷж ·еҶҷпјҡ

      def explore_array(method, *arguments)
  ['a','b','c'].send(method, *arguments)
end

explore_array(:include?, 'a') #= > true
explore_array(:size) #= > 3

  • жұЎжҹ“еҜ№иұЎе’Ңе®үе…Ёзә§еҲ«

    й’ҲеҜ№evalеӯҳеңЁзҡ„е®үе…Ёй—®йўҳпјҢеҸҜд»ҘйҮҮз”ЁдёҖдәӣдҪҝе®ғеҸҳеҫ—е®үе…Ёзҡ„ж–№жі•гҖӮ

    RubyдјҡиҮӘеҠЁжҠҠдёҚе®үе…Ёзҡ„еҜ№иұЎж Үи®°дёәжұЎжҹ“еҜ№иұЎпјҢдҪ еҸҜд»ҘйҖҡиҝҮз»ҷ$SAFEиөӢеҖјпјҲ[0,1, 2,3]пјүжқҘи®ҫзҪ®е®үе…Ёзә§еҲ«пјҢзҰҒжӯўжҹҗдәӣжҪңеңЁзҡ„еҚұйҷ©ж“ҚдҪңпјҢе…¶дёӯ0жңҖдҪҺпјҢ3жңҖй«ҳгҖӮ

    йҖҡиҝҮдҪҝз”Ёе®үе…Ёзә§еҲ«пјҢеҸҜд»Ҙдёәevalж–№жі•еҲӣйҖ дәҶдёҖдёӘеҸҜжҺ§зҡ„зҺҜеўғпјҢеғҸиҝҷж ·зҡ„зҺҜеўғз§°д№ӢдёәжІҷзӣ’гҖӮ

з”ЁevalжқҘе®ҡд№үadd_checked_attr

гҖҢе…¶е®һпјҢиҝҷйҮҢжҲ‘е№¶дёҚжҳҜеҫҲзҗҶи§ЈпјҢдёәд»Җд№ҲиҰҒи°ғз”ЁevalжқҘе®ҡд№үиҝҷдёӘmethodпјҢз„¶еҗҺеҸҲеҺ»жҺүе®ғпјҹдёәд»Җд№ҲдёҚзӣҙжҺҘз”Ёclass_evalзҡ„ж–№ејҸжқҘе®ҡд№үпјҹйҡҫйҒ“иҝҷж ·з»•дёҖеңҲпјҢд»…д»…жҳҜдёәдәҶеҗ‘иҜ»иҖ…д»Ӣз»ҚevalпјҹгҖҚ

дёҚеҗҗзіҹпјҢзӣҙжҺҘзңӢи§Јзӯ”пјҡ

def add_checked_attr(klass, attr)
  eval "
  class #{klass}
    def #{attr}=(value)
      raise 'Invalid attribute' unless value
      @#{attr} = value
    end
    def #{attr}
      @#{attr}
    end
  end
  "
end

еҺ»жҺүevalпјҢеҠ дёҠblock

йүҙдәҺevalзҡ„й—®йўҳпјҢж”№з”Ёclass_evalжқҘжү“ејҖзұ»пјҢеҗҢж—¶з»ҷж–№жі•ж·»еҠ дёҖдёӘblock

def add_checked_attr(klass, attr, &validation)
  klass.class_eval do

    define_method "#{attr}=" do |value|
      raise 'Invalid attribute' unless validation.call(value) # validation жҳҜдёҖдёӘproc
      instance_variable_set("@#{attr}", value)
    end

    define_method attr do
      instance_variable_get("@#{attr}")
    end

  end
end

ж”№йҖ жҲҗзұ»е®Ҹ

дёәдәҶи®©add_checked_attrеҜ№жүҖжңүзұ»йғҪеҸҜз”ЁпјҢеҸҜд»Ҙе°Ҷе®ғе®ҡд№үеңЁclassдёӯпјҢеҸҰдҝ®ж”№ж–№жі•еҗҚдёәattr_checkedпјҢеҺ»жҺүзұ»еҸӮж•°гҖӮ

и§Јзӯ”еҰӮдёӢпјҡ

class Class
  def attr_checked(attr, &validation)
    define_method "#{attr}=" do |value|
      raise 'Invalid attribute' unless validation.call(value) # validation жҳҜдёҖдёӘproc
      instance_variable_set("@#{attr}", value)
    end

    define_method attr do
      instance_variable_get("@#{attr}")
    end
  end
end

жңҖеҗҺиҜ·дёҠжҲ‘д»¬зҡ„hooksгҖӮ

й’©еӯҗж–№жі•

зңӢдёӘдҫӢеӯҗпјҡ

module M
  def self.included(othermod)
    puts "M was included into #{othermod}"
  end
end

class C
  include M
end
#= > M was included into C

Module#includedеңЁдёҖдёӘmoduleиў«includeж—¶пјҢиў«иҮӘеҠЁи°ғз”ЁпјҢиҝҷз§Қж–№жі•з§°д№Ӣдёәй’©еӯҗж–№жі•пјҢеӣ дёәе®ғд»¬еғҸй’©еӯҗдёҖж ·пјҢй’©дҪҸдёҖдёӘзү№е®ҡзҡ„дәӢд»¶гҖӮзұ»дјјзҡ„ж–№жі•иҝҳжңүModule#method_added, Module#extend_objectзӯүгҖӮ

жғіиҰҒжҚ•иҺ·еҚ•д»¶ж–№жі•зҡ„й’©еӯҗдәӢд»¶пјҢйңҖиҰҒдҪҝз”ЁBasicObjectзҡ„singleton_xxxж–№жі•пјҢжҜ”еҰӮsingleton_method_addedгҖӮ

й’ҲеҜ№дёҠйқўзҡ„дҫӢеӯҗпјҢеҸҜд»ҘиҰҶеҶҷincludeж–№жі•еҫ—еҲ°дёҖж ·зҡ„ж•Ҳжһңпјҡ

module M;end

class C
  def self.include(*modules)
    puts "#{modules} was included into C"
    super
  end
  include M
end
#= > [M] was included into C

жіЁж„ҸиҝҷйҮҢиҰҶеҶҷincludeдёӯпјҢжңҖеҗҺи°ғз”ЁдәҶsuperпјҢдёҚз„¶иҝҷдёӘmoduleжҳҜдёҚдјҡиў«includeзҡ„гҖӮиҝҷз§ҚйҖҡиҝҮиҰҶеҶҷж·»еҠ йўқеӨ–еҠҹиғҪзҡ„ж•ҲжһңпјҢд№ҹеҸҜд»ҘйҖҡиҝҮзҺҜз»•еҲ«еҗҚзҡ„ж–№ејҸе®һзҺ°гҖӮ

еҢ…еҗ«е№¶жү©еұ•зұ»зҡ„ж–№ејҸе°ұз”ЁеҲ°дәҶй’©еӯҗпјҢзңӢдёӘдҫӢеӯҗпјҡ

module M
  def self.included(klass)
    klass.extend ClassMethods
  end
end

module ClassMethods
  def my_method
    puts "it could be a class method"
  end
end

class D
  include M
end

D.my_method #= > it could be a class method

зұ»DеҢ…еҗ«дәҶжЁЎеқ—MпјҢи°ғз”ЁдәҶMзҡ„й’©еӯҗж–№жі•:includedпјҢ иҝҷдёӘй’©еӯҗж–№жі•жҺҘзқҖз”Ёmodule ClassMethodsжү©еұ•дәҶзұ»DпјҢдҪҝеҫ—module ClassMethodsзҡ„my_method жҲҗдёәдәҶзұ»Dзҡ„зұ»ж–№жі•гҖӮ

RailsдёӯActiveSupportжңүдёҖдёӘmodule ConcernпјҢе°ұе°ҒиЈ…дәҶеҢ…еҗ«е№¶жү©еұ•зҡ„жҠҖе·§гҖӮ

дҪҝз”Ёиҝҷз§ҚжҠҖе·§пјҢе®ҢжҲҗжңҖеҗҺдёҖжӯҘгҖӮ

module CheckedAttributes
  def self.included(base)
    base.extend ClassMethods
  end

  module ClassMethods
    def attr_checked(attr, &validation)
      define_method "#{attr}=" do |value|
        raise 'Invalid attribute' unless validation.call(value) # validation жҳҜдёҖдёӘproc
        instance_variable_set("@#{attr}", value)
      end

      define_method attr do
        instance_variable_get("@#{attr}")
      end
    end
  end
end
