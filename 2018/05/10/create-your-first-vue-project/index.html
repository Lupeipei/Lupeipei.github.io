其中dist是打包后生成的，即执行npm run build后生成的。

简单看看各个目录放的是什么。

  • build/：打包相关
  • config/：配置文件存放目录
  • node_modules/: 项目中用到的第三方包，特别大，安装所有的第三方包，执行 npm install
  • src/：源代码所在目录，也是我们主要活动的地方，你可以将它看成是rails的app，一般在src/components/下面添加组件文件，然后在src/router/index.js中定义新的路径。后面我们会建一个新的组件试试。
  • static/：静态文件
  • test/：测试
