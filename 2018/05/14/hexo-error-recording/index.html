解决方法

按照提示，使用npm rebuild ，rebuild过程中出现了各种报错，最后还输出了这么一句This is probably not a problem with npm, 不是npm，难道是hexo?

随后Google，在hexo issues里找到了这篇#2534。尝试着用里面提到的方法：

npm install时出错，后面我又npm install了一次，OK，没有报错，但是奇怪的是执行hexo g -d 依然会出现上面的报错，说明问题还是没有解决。

没有报错，应该是OK了，稍稍踏实了，再次执行hexo g -d，结果 还是报错，比上次更为严重：

hexo已经挂了。

显示Cannot find module 'highlight.js/lib/languages/shell' .

既然找不到highlight， 那就安装一个吧！终端执行：

出现报错：

发现fsevents在之前的报错中也有出现过，难道是fsevents与node版本不兼容？Google后找到了这篇：

Install on OS X fails as prebuilt binary seems to be missing from S3

上面提到了1.1.1版本存在bug，升级到1.2+，好，死马当活马医，npm i fsevents ，装了最新的fsevents, 1.2.3.

终端再次执行hexo g -d， OK，没有出现上诉报错了，一切正常!
