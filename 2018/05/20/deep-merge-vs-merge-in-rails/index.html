RailsдёӯmergeдёҺdeep_mergeзҡ„е·®ејӮ

еҸ‘иЎЁдәҺ 2018-05-20 | еҲҶзұ»дәҺ CS |   |   784 words

Rails е°ҸзҷҪзҡ„еЎ«еқ‘д№Ӣи·ҜгҖӮ

еҶҷеңЁеүҚйқў

еүҚеҮ еӨ©иў«иҖҒеӨ§й—®еҲ°mergeдёҺdeep_mergeзҡ„еҢәеҲ«пјҢдёҖж—¶иҜӯеЎһпјҢиҜ•дәҶеҮ дёӘдҫӢеӯҗпјҢеӨ§иҮҙgetеҲ°дәҶдёӨиҖ…зҡ„еҢәеҲ«пјҢдҪҶжҳҜеҪ“ж—¶жІЎжңүи®°еҪ•дёӢжқҘпјҢиҝҷйҮҢеҒҡдёҖдёӢз¬”и®°пјҢеҠ ж·ұеҚ°иұЎгҖӮ

жӯЈж–Ү

  • mergeз”Ёжі•

    еңЁdocsдёӯжҗңзҙўmergeпјҢеҸҜд»ҘзңӢеҲ°жңү10дёӘеҗҚдёәmergeзҡ„methodпјҢиҝҷйҮҢжҲ‘д»¬иҰҒеҜ№жҜ”deep_mergeпјҢжүҖд»ҘйҖүжӢ©з»§жүҝиҮӘHashзҡ„ActiveSupport::HashWithIndifferentAccessдёӢзҡ„method mergeгҖӮ

    йҖүиҮӘActiveSupport::HashWithdifferentAccess

    merge(hash, &block)

    This method has the same semantics of update, except it does not modify the receiver but rather returns a new hash with indifferent access with the result of the merge.

    жӯӨеӨ„зҡ„mergeдёҚдјҡдҝ®ж”№еҪ“еүҚеҜ№иұЎпјҢиҖҢжҳҜе°ҶеҗҲе№¶еҗҺзҡ„з»“жһңд»ҘдёҖдёӘж–°зҡ„hashиЎЁж јејҸиҝ”еӣһгҖӮ

    е…¶е®һиҝҷйҮҢзҡ„mergeе°ұзӯүеҗҢдәҺrubyдёӯзҡ„mergeпјҢ еҜ№дәҺдёӨдёӘhashпјҢй’ҲеҜ№е…ұжңүзҡ„keyпјҢй»ҳи®ӨеҸ–еҗҺдёҖдёӘhashзҡ„keyеҜ№еә”зҡ„valueгҖӮ

    зңӢдёӘдҫӢеӯҗпјҡ

    h1 = { "a" => 100, "b" => 200 }
h2 = { "b" => 254, "c" => 300 }
h1.merge(h2) #=> {"a"=>100, "b"=>254, "c"=>300}

    дҪ еҸҜд»ҘйҖҡиҝҮblockжқҘжҢҮе®ҡmergeзҡ„и§„еҲҷпјҡ

    h1 = { "a" => 100, "b" => 200 }
h2 = { "b" => 254, "c" => 300 }
h1.merge(h2) {|k,v1,v2| v1+v2} #=> {"a"=>100, "b"=>454, "c"=>300}
h1.merge(h2) {|k,v1,v2| v1} #=> { "a" => 100, "b" => 200 }

  • deep_mergeз”Ёжі•

    е®ҳж–№з»ҷеҲ°зҡ„и§ЈйҮҠе’ҢдҫӢеӯҗжҳҜиҝҷж ·зҡ„пјҡ

    deep_merge(other_hash, &block)

    Returns a new hash with self and other_hash merged recursively.

    иҝ”еӣһдёҖдёӘж–°зҡ„hashпјҢ е°ҶеҪ“еүҚеҜ№иұЎе’Ңе…¶д»–зҡ„hashиҝӣиЎҢйҖ’еҪ’mergeгҖӮ

    иҝҳжҳҜд»ҘдёҠйқўзҡ„дҫӢеӯҗдёәдҫӢпјҡ

    h1 = { "a" => 100, "b" => 200 }
h2 = { "b" => 254, "c" => 300 }
h1.deep_merge(h2) #=> {"a"=>100, "b"=>254, "c"=>300}

    еҗҢж ·пјҢдҪ д№ҹеҸҜд»ҘйҖҡиҝҮblockжқҘжҢҮе®ҡdeep_mergeзҡ„и§„еҲҷпјҡ

    h1 = { "a" => 100, "b" => 200 }
h2 = { "b" => 254, "c" => 300 }
h1.deep_merge(h2) {|k,v1,v2| v1+v2} #=> {"a"=>100, "b"=>454, "c"=>300}
h1.deep_merge(h2) {|k,v1,v2| v1} #=> { "a" => 100, "b" => 200 }

  • дәҢиҖ…е·®ејӮ

    д»ҺдёҠйқўзҡ„дҫӢеӯҗзңӢпјҢдјјд№ҺдёӨиҖ…жІЎжңүд»Җд№ҲеҢәеҲ«гҖӮ

    еңЁйқһеөҢеҘ—зҡ„hashиЎЁдёӯпјҢmergeиғҪеҒҡзҡ„пјҢdeep_mergeд№ҹиғҪеҒҡгҖӮ

    дҪҶжҳҜй’ҲеҜ№nested hashпјҢдёӨиҖ…е°ұдёҚдёҖж ·дәҶгҖӮ

    зңӢдёӘдҫӢеӯҗпјҡ

    h1 = { a: true, b: { c: [1, 2, 3], x: [1, 2, 3] } }
h2 = { a: false, b: { x: [3, 4, 5] } }
h1.merge(h2) #=> {:a=>false, :b=>{:x=>[3, 4, 5]}}
h1.deep_merge(h2) #=> {:a=>false, :b=>{:c=>[1, 2, 3], :x=>[3, 4, 5]}}

    дҪҝз”Ёmergeж—¶пјҢеҜ№дәҺзӣёеҗҢзҡ„key, е®ғдёҚдјҡеҺ»жЈҖжҹҘеҜ№еә”зҡ„valueжҳҜдёҚжҳҜдёҖдёӘhashпјҢе°ұжҳҜз®ҖеҚ•жҢүз…§mergeзҡ„и§„еҲҷиҝ”еӣһvalueгҖӮ

    дҪҝз”Ёdeep_mergeпјҢдјҡжЈҖжҹҘеҗҢж ·зҡ„keyпјҢеҜ№еә”зҡ„valueжҳҜдёҚжҳҜйғҪжҳҜhashпјҢеҰӮжһңжҳҜ, еҲҷеҜ№иҝҷдёӨдёӘhashеҶҚж¬ЎиҝӣиЎҢdeep_mergeпјҢд№ҹе°ұжҳҜиҝ”еӣһ{ c: [1, 2, 3], x: [1, 2, 3] }.deep_merge{ x: [3, 4, 5] }зҡ„з»“жһңпјҢд№ҹе°ұжҳҜжҰӮеҝөдёӯжҸҗеҲ°зҡ„йҖ’еҪ’mergeгҖӮ

    еҶҚзңӢдёӘдҫӢеӯҗпјҡ

    h1 = { a: true, b: { c: [1,2,3], x: { y: 1 } } }
h2 = { a: false, b: { x: { y: 2, z: 1} } }
h1.deep_merge(h2) #=> {:a=>false, :b=>{:c=>[1, 2, 3], :x=>{:y=>2, :z=>1}}}

    д»ҺдёҠйқўиҝҷдёӘдҫӢеӯҗеҸҜд»ҘзңӢеҮәпјҢй’ҲеҜ№е…ұжңүзҡ„keyпјҢdeep_mergeдјҡдёҖзӣҙжЈҖжҹҘеҜ№еә”зҡ„valueжҳҜеҗҰйғҪжҳҜhashпјҢеҰӮжһңжҳҜпјҢеҲҷз»§з»ӯеҜ№hashиҝӣиЎҢdeep_mergeпјҢзӣҙеҲ°жІЎжңүhashиҝӣиЎҢmergeдёәжӯўгҖӮ

    е°Ҹз»“

  • й’ҲеҜ№йқһеөҢеҘ—зҡ„hashпјҡmergeе’Ңdeep_mergeе№¶жІЎжңүд»Җд№Ҳе·®ејӮгҖӮ

  • й’ҲеҜ№еөҢеҘ—зҡ„hashпјҡ й’ҲеҜ№зӣёеҗҢзҡ„keyпјҢmergeдёҚдјҡжЈҖжҹҘпјҢзӣҙжҺҘеӨ„зҗҶпјҢж №жҚ®mergeи§„еҲҷиҝ”еӣһvalueпјҢdeep_mergeеҲҷдјҡдёҖзӣҙжЈҖжҹҘеҜ№еә”зҡ„valueжҳҜеҗҰйғҪжҳҜhashпјҢеҰӮжһңжҳҜпјҢеҲҷз»§з»ӯеҜ№hashиҝӣиЎҢdeep_mergeпјҢзӣҙеҲ°жІЎжңүhashиҝӣиЎҢmergeдёәжӯўгҖӮ
